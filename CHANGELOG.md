# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-02-01

### Added
- Initial release of justfile templates category
- Python project template with UV, Ruff, MyPy, Pytest
- FastAPI template with Uvicorn, Alembic, Docker
- CLI application template with packaging and publishing
- ML/Data Science template with Jupyter, MLflow, DVC
- General minimal starter template
- Comprehensive documentation for all templates
- GitHub Actions CI/CD for template validation
- Contributing guidelines
- MIT License

### Documentation
- Main repository README
- Category README for justfile templates
- Individual README for each template
- Getting started guide
- Customization guide
- Best practices guide

### Infrastructure
- GitHub Actions workflow for validation
- Issue templates
- Pull request template
- Code of Conduct
